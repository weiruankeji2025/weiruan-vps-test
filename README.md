📖 简介 (Introduction)
威软科技 VPS 测试脚本 是一款基于 Shell 编写的轻量级检测工具。它能快速检测您的 Linux 服务器对全球主流流媒体平台的解锁情况。

v5.1 版本采用了全新的 “点阵对齐 (Dot Leaders)” 排版引擎，彻底解决了传统脚本在不同终端下竖线错位的问题，配合霓虹色系 UI，带来赏心悦目的测试体验。

⚡ 一键使用 (Quick Start)
无需任何依赖，直接复制下方命令到终端即可运行：

bash <(curl -L -s https://raw.githubusercontent.com/weiruankeji2025/weiruan-vps-test/main/weiruan.sh)
(支持 Debian / Ubuntu / CentOS / Alpine 等主流 Linux 发行版)

✨ 核心特性 (Features)
🎨 赛博朋克 UI：采用荧光绿、亮黄、青色构建的高对比度视觉体验，清晰直观。

📏 完美排版引擎：摒弃传统表格竖线，采用 Name ........... Status 点阵自适应对齐，强迫症福音。

🌍 全区域覆盖：内置 120+ 个流媒体检测项，涵盖北美、亚洲、欧洲三大核心区域。

🤖 智能模拟：内置真实浏览器 User-Agent，有效解决 Paramount+ 等网站报 406 错误的问题。

📊 全网动态统计：接入 CountAPI，精准记录全网累计测试次数，精确到个位。

🚀 极速响应：基于 curl 核心检测，无臃肿 Python/Node 依赖，秒级启动。
